# 📱 제품명: 오늘레벨업 (가칭) PRD

## 1. 🎯 제품 개요
모든 행동과 보상의 기준을 '포모도로(시간 블록)'와 '포인트(재화)'로 규격화한 자기 주도형 습관 형성 앱입니다. 단일 기기 내에서 **어린이 모드**와 **부모 관리 모드(PIN 잠금)**를 지원하며, 아이는 미션을 통해 포인트를 얻고, 부모가 세팅한 상점에서 온/오프라인 보상 카드를 구매하여 소비하는 작은 경제 시스템을 경험합니다.

## 2. 🎯 비즈니스 및 사용자 목표
* **Primary Goal:** 아이가 자발적으로 타이머를 켜는 습관을 만들고, '내가 노력해서 얻은 포인트'로 보상을 획득하는 자립적 가치관 형성.
* **Success Metric:**
  * 일일 포모도로 사이클 완료 횟수 (미션 + 보상 포함)
  * 포인트 상점 재화 순환율 (Earn & Burn 비율)
  * 다중 자녀 환경에서의 계정별 활성도 유지

## 3. 👤 타겟 사용자
* **Primary (사용자):** 초등학생 ~ 중학생
* **Secondary (관리자):** 부모 (다중 자녀 프로필 전환 및 개별 목표/보상 관리 지원)

## 4. 🔑 핵심 가치 및 철학
* **시간의 규격화:** 1포모도로(예: 25분) = 1미션 완료 = 1디지털 보상 소비.
* **통제된 주도권:** 부모가 안전한 울타리(보상 상점, 그랜드 퀘스트)를 제공하고, 아이는 그 안에서 자유롭게 선택.
* **보상의 3단계 다층화:** [단기] 포인트 획득 ➡️ [중기] 레벨업 잭팟 ➡️ [장기] 그랜드 퀘스트 실물 보상.

## 5. 🧩 핵심 기능 (MVP Scope)

### 5.1 🃏 카드 기반 경제 시스템 (Card & Economy)
* **일일 퀘스트 (Daily Quest):** 부모가 생성한 퀘스트(예: 수학 문제 풀기, 영어 단어 암기)를 아이가 선택하여 수행. 퀘스트 선택 시 포모도로 집중 타이머가 자동 실행되며, 완료 시 기본 포인트 + 퀘스트별 보너스 포인트 지급. 각 퀘스트는 하루 반복 가능 횟수가 부모에 의해 제한됨.
* **포인트 상점 (Shop):** 아이가 획득한 포인트로 부모가 진열해둔 '보상 카드' 구매.
* **지갑 (Inventory):** 구매한 보상 카드를 보관하고 원할 때 꺼내어 사용.

### 5.2 ⏳ 듀얼 포모도로 타이머 (Dual Timer)
* **집중 타이머 (Work):** 미션 카드 실행 시 작동. 방해금지 UI, 중도 포기 시 보상 없음.
* **플레이 타이머 (Play):** 시간형 보상 카드(게임/유튜브 등) 사용 시 작동. 지정된 시간 경과 후 강력한 종료 알림(Time Over).

### 5.3 🔒 부모 관리 모드 (Parent Admin)
* **보안 게이트:** 4자리 PIN 입력 시에만 진입 가능.
* **하루 일당 포인트 (Daily Allowance):** 부모가 설정한 포인트를 매일 기본 지급 (디폴트 30P). 아이는 홈 화면에서 '오늘의 포인트 받기' 버튼으로 수령. 미션 수행 없이도 최소한의 포인트를 보장하여 경제 시스템 참여 동기 부여.
* **일일 퀘스트 관리:** 퀘스트 이름, 보상 포인트, 하루 반복 가능 횟수, 아이콘을 설정하여 아이의 홈 화면에 퀘스트 카드로 진열. 활성화(ON/OFF) 제어 가능.
* **상점 관리:** 디지털 시간형 카드(환율 설정) 및 실물 소멸형 카드(과자, 소원권 등) 생성, 가격 책정, 활성화(ON/OFF) 제어.
* **그랜드 퀘스트 관리:** "수학 문제집 완권", "레벨 10 달성" 등 장기 목표 설정 및 달성 시 수동 승인.
* **실물 보상 승인:** 아이가 오프라인 보상(라면, 과자 등) 사용 시 부모가 확인 후 카드 소멸 처리.

## 6. 📊 핵심 데이터 모델 (ERD 기초)

### 6.1 User & Auth
* `account_id` (Parent Account)
* `child_profile_id` (Profile for each child)
* `pin_hash` (Parent authentication)
* `current_level`, `current_exp`, `point_balance`

### 6.2 Quest & Items
* `grand_quest_id`: 장기 목표 (status: ONGOING, ACHIEVED, REWARDED)
* `shop_item_id`: 부모가 등록한 상점 상품
  * `type`: TIMER (디지털 시간), CONSUMABLE (실물 오프라인)
  * `cost_points`: 가격
  * `is_active`: 상점 노출 여부

### 6.3 Transactions
* `inventory_id`: 아이가 구매하여 보유 중인 카드
* `session_log`: 타이머 수행 기록 (work/play 시간 트래킹)
